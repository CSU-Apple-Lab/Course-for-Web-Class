# 1.2 关于课程
----
---
## 面向对象
* 想学习web开发，从零开始的15、16童鞋为主

## 学习目的
* 了解web世界，理解一些所用到的协议
* 掌握web开发的语言
* 熟悉擅长一些开发框架
* 了解web架构，了解服务器运维
* 能独立完成网站应用的前端或者后端的开发

## 最终目的
* 能够独立或合作完成一套中小web应用的开发
* 配合移动开发组完成接口开发
* 让我们实验室更好的发展，形成开源极客的开发氛围
